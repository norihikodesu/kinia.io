Kinia（キニア）

気になった時に、そっと見守る

家族の帰りが遅いとき、ふと「どうしたのかな」と気になる瞬間。
Kiniaはそんな時だけ、家族の様子をそっと確認できるアプリです。
常に監視し合うのではなく、通知エリアへの出入りだけをお知らせ。
必要な時だけ、ゆるくつながる安心を。

Story

なぜKiniaを作ったのか

私は実家の家族とも義理の家族とも、比較的仲が良い方だと思っています。
これまでは別の位置共有アプリを使ってきました。

ただ、常にお互いの位置が詳細に見えるという関係は、
共有のレベルとして少し強すぎるのではないか — そう感じるようになりました。

[画像: 家族の穏やかなイラスト — images/story_family.webp]

適切な距離感を保てている間柄であっても、常時の位置共有はギリギリのライン。
自分の実家の両親にいたっては、心理的ハードルが高すぎて
アプリの紹介すらできていませんでした。

「今、家にいるのかどうか」—— そのくらいの情報を、
お互いにもっと緩く共有できたら、
より適切な距離感で付き合えるのではないか。

監視するのもされるのも、お互いに嫌なもの。
「子どもの見守り」ではなく「家族同士のつながり」を考えたとき、
もう少し距離を取った「ゆるいつながりの形」があるのではないか。
それがKiniaの開発のきっかけでした。

Problem

既存の位置共有アプリが抱える問題

調べていくうちに、安全上の理由から「子どもの見守り」には需要があり、
そこでは詳細な位置共有が必要なことも分かりました。
しかし、そこにはニーズの非対称性が存在します。

📡

監視する側の視点

常にリアルタイムで詳細な情報を見たい。でも自分の情報は見せたくない。

😣

見守られる側の負担

常に位置を見られている感覚はストレス。昔はこれほど詳細に干渉する術がなかった。

⚖️

不公平な非対称性

見る側と見られる側で共有レベルが異なる。一方的な関係になりがち。

🚫

高すぎる心理的ハードル

親世代に「位置を常に共有しよう」とは言い出しにくい。導入自体が難しい。

[画像: 監視vs見守りの対比 — images/problem_contrast.webp]

Solution

Kiniaの答え

見守られる側が負担に感じない共有のされ方。
そして「いざという時」だけはしっかりと繋がれる仕組み。

  • ✓ 初期設定は通知エリアのみ — 設定するまで何も共有されない
  • ✓ 「自宅を出ました」「学校にいます」レベルの共有
  • ✓ 地図ではなくテキスト一覧がメイン画面
  • ✓ 必要な時だけ、一時的に詳細な位置を共有
  • ✓ お子さんの自立性と個性を尊重した設計
[画像: テキストベースUI — images/ui_member_list.webp]

How it works

届く通知は、これだけ

詳細な現在地ではなく、登録したエリアへの出入りだけ。
誰が、いつ、どこにいるか — それだけで十分な安心があります。

Kinia 通知
📍 【IN】お母さん → 「自宅」
📍 【OUT】太郎 → 「学校」
📍 【IN】太郎 → 「塾」
📍 【OUT】お父さん → 「会社」

Use cases

あなたの家族に合った使い方

👨‍👩‍👧

子どもの見守り

学校・塾・自宅を通知エリアに設定。
「塾に着いたよ」の通知で安心。
常に位置を追わなくても大丈夫。

👴👵

離れて暮らす親

実家を通知エリアに設定するだけ。
「今日も家にいるな」が分かる。
心理的ハードルゼロで始められる。

🤝

待ち合わせ

一時共有で6時間だけ詳細位置をON。
お互いの距離が分かって便利。
終わったら自動で元に戻る。

🌍

すべてを共有したい家族

もちろん従来の位置共有も可能。
地図・軌跡・速度すべて確認。
共有レベルは家族の形に合わせて。

Vision

Kiniaが目指す未来

🆘

災害時の安否確認

将来的には、災害が起こった時にお互いの安否確認や

普段は「ゆるいつながり」、いざという時は「確実につながる」。
それがKiniaの目指す未来です。

Name

「Kinia（キニア）」の由来

気にかける（Kini）＋近く（Near）

家族の帰りが遅いとき、ふと「どうしたのかな」と気になった（Kini）瞬間、
アプリを開けば家族の存在を近く（Near）に感じられる。

常に監視し合うのではなく、気になった時だけそっと見守る。
「緩やかで温かい距離感」を名前に込めました。

ゆるいつながり、始めませんか

Kiniaは現在ベータテスト中です。
あなたの家族の「ちょうどいい距離感」を、一緒に見つけましょう。
