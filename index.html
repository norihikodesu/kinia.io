Story

なぜKinia（キニア）を作ったのか

実家の家族とも、義理の家族とも、それなりにうまくやってきたと思います。離れて暮らしていても、お互いの存在を身近に感じていたくて、家族みんなで位置共有アプリを使っていました。

でも、使い続けるうちに、小さな違和感が積もっていきました。

まず、距離感が近すぎること。常にお互いの居場所が詳細に見える関係は、信頼し合っている間柄であっても、どこか息苦しい。自分の行動が逐一把握されていると思うだけで、何でもないはずの外出にも、うっすらとした緊張感がまとわりつく。

そして、共有されている情報が多すぎること。リアルタイムの現在地、移動中の速度、バッテリーの残量——家族が安心するために、そこまでの情報が本当に必要なのだろうか。知りたいのは「元気かどうか」であって、「今どの道を何キロで走っているか」ではないはずです。

その一方で、離れて暮らす実家の両親とは、もっとつながっていたいとも感じていました。元気にしているか。ちゃんと外に出られているか。そういうことを、ゆるく知っておきたい。

ところが、既存の位置共有アプリを両親に勧めようとすると、途端にハードルが上がります。「あなたの居場所を常に見られるようにして」——仲の良い親子であっても、そんなお願いはなかなかできません。

つながりたいのに、つながる手段そのものがハードルになっている。そのもどかしさが、ずっと心のどこかにありました。

「今、家にいるのかどうか」——そのくらいの情報を、もっと気軽に共有できたら。お互いの日常を尊重しながら、それでもちゃんとつながれる方法はないだろうか。

監視するのも、されるのも、誰だって嫌なものです。「子どもを管理する」のではなく、「家族としてつながる」ことを考えたとき、今よりもう少しだけ距離をとった、ゆるいつながりの形があっていいのではないか。Kiniaは、そこから生まれました。

Problem

既存の位置共有アプリが抱える問題

開発を始めてから改めて調べてみると、安全上の理由から「子どもの見守り」には確かに大きな需要がありました。そして、そのためには詳細な位置情報の共有が必要な場面もあることが分かりました。

しかし、既存のアプリには構造的な問題があります。それは「見る側」と「見られる側」のニーズが対称ではないということです。

📡

見る側の心理

親は子どもの位置を常に詳しく知りたい。けれど、自分の位置は子どもにそこまで見せたくない。見守りたいけれど、見守られたくはない。

😣

見られる側の負担

24時間、位置を見られている感覚はストレスです。ほんの一世代前まで、こんな詳細に家族の行動を把握する術はありませんでした。便利な技術が、新しい息苦しさを生んでいます。

⚖️

不公平な関係

見る側と見られる側で、共有のレベルが違う。この一方通行の関係は、家族の中にある信頼を、気づかないうちに少しずつ削っていきます。

🚫

導入のハードル

「位置を常に共有して」とは、家族にすら言い出しにくい。導入のハードルが高すぎて、本当に見守りが必要な人——たとえば離れて暮らす高齢の親——にこそ、届いていません。

もちろん、現代は子どもが犯罪に巻き込まれるリスクもあり、安全のために位置を把握したい気持ちはよく分かります。見守ること自体が悪いわけではありません。

問題は「見守り方」にあるのではないか。見守られる側が負担に感じない共有のあり方と、いざという時にはしっかりとつながれる仕組み——その両立を目指したのが、Kiniaのコンセプトです。

Concept

全世代に向けた「つながりの形」

子どもの自立心を尊重しながら親は安心したい。子世代は離れた親の暮らしを気にかけたい。夫婦はお互いの行動を干渉せずに、でも帰りが遅ければ心配したい。——家族の「つながりたい」には、世代ごとにまったく違う形があります。

Kiniaでは、アプリを入れた時点での位置共有を通知エリアの出入りだけに限定しています。「自宅」「学校」「会社」など、あらかじめ設定した場所への出入りだけが家族に届く。それ以外の行動は、誰にも見えません。

家族に届く通知の例
🏠 【OUT】太郎 → 自宅を出ました
🏫 【IN】太郎 → 学校に着きました
📚 【OUT】太郎 → 塾を出ました
🏠 【IN】太郎 → 自宅に着きました

メイン画面は地図ではなく、テキストベースのメンバー一覧です。「誰が、いつ、どこにいて、元気なのか」が一目で分かる。位置共有アプリとしては少し変わった設計ですが、これがKiniaの考える「ちょうどいい距離感」です。

もちろん、もっと詳しくつながりたい家族のために、すべての位置情報を共有する設定も用意しています。家族の形は一つではないから、つながり方も一つである必要はありません。

Solution

Kiniaの答え

見守られる側が負担に感じない共有のされ方。
そして「いざという時」だけはしっかりと繋がれる仕組み。

  • ✓ 初期設定は通知エリアのみ — 設定するまで何も共有されない
  • ✓ 「自宅を出ました」「学校にいます」レベルの共有
  • ✓ 地図ではなくテキスト一覧がメイン画面
  • ✓ 必要な時だけ、一時的に詳細な位置を共有
  • ✓ お子さんの自立性と個性を尊重した設計

Security

運営すら位置情報を見られない設計

Kiniaでは、サーバーに保存される位置情報は独自の暗号化技術でオフセット処理されています。 万が一データベースが流出しても、実際の座標を復元することは不可能です。

🔐

座標オフセット暗号化

全ての位置情報は、端末にのみ保存される「合言葉」から導出したオフセットでシフトされてからサーバーに保存されます。合言葉はサーバーに送信されないため、運営を含め誰も実際の位置を逆算できません。

👤

個人名をサーバーに保存しない

サーバー上にはランダムなIDのみが保存されます。家族の表示名はあなたの端末にだけ保存され、サーバーには送信されません。データベースを見ても「誰が誰か」は分かりません。

🔑

グループIDのハッシュ化

グループへの参加に使う合言葉も、暗号学的ハッシュ関数で一方向変換されてからサーバーに保存されます。サーバーから合言葉を逆算することは不可能です。

🛡️

エンドツーエンドの安全性

位置情報が意味を持つのは、合言葉を知っている家族の端末上だけです。サーバーは暗号化されたデータを中継するだけの存在です。

More Features

位置共有だけじゃない、家族のための機能

👟

歩数と元気度

家族の今日の歩数と、そこから算出した「元気度」が分かります。位置を共有しなくても「今日も元気に動いてるな」と安心できます。

💬

ステータスコメント

「会議中」「外出中」「帰宅中」など、一言メッセージを設定できます。有効時間を選べるので、自動で消えます。

🆘

💰

完全無料・広告なし

基本機能はすべて無料で利用できます。広告表示もありません。家族の安心にお金はかかりません。

Name

「Kinia（キニア）」の由来

気にかける（Kini）＋近く（Near）

家族の帰りが遅いとき、ふと「どうしたのかな」と気になった（Kini）瞬間、
アプリを開けば家族の存在を近く（Near）に感じられる。

常に監視し合うのではなく、気になった時だけそっと見守る。
「緩やかで温かい距離感」を名前に込めました。

ゆるいつながり、始めませんか

Kiniaは現在ベータテスト中です。
あなたの家族の「ちょうどいい距離感」を、一緒に見つけましょう。
