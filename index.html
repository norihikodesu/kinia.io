Story

なぜKinia（キニア）を作ったのか

私の家族は実家とも義理の家族とも、それなりにうまくやってきたと思います。離れて暮らしていても、お互いが都合のいい時に声をかけやすいように、家族みんなで位置共有アプリを使っていました。

でも、使い続けるうちに、小さな違和感が積もっていきました。

まず、距離感が近すぎること。常にお互いの居場所が詳細に見える関係は、信頼し合っている間柄であっても、どこか息苦しい。自分の行動が逐一把握されていると思うだけで、何でもないはずの外出にも、うっすらとした緊張感がまとわりつく。

そして、共有されている情報が多すぎること。リアルタイムの現在地、移動中の速度、バッテリーの残量——家族が安心するために、そこまでの情報が本当に必要なのだろうか。知りたいのは「元気に過ごしているか」であって、「今どの道を何キロで走っているか」ではないはずです。

その一方で、実家の両親とはあまり連絡が取れていないと感じていました。元気でいるのかというのも、なんだか声をかけづらい。どのくらいの頻度で外出しているのか、2人とも元気に過ごしているのかを知る術があまりありませんでした。

確かに連絡を取ればいいのかもしれません。でも、それが難しいと感じているのも事実です。実家の状況を、緩く、適切な距離感で把握していたい——そう感じていました。

両親も高齢になり、いろいろな心配がつきまといます。しかし、両親からは「監視されるのはとても嫌だ」「干渉はしてほしくない」と言われています。

ところが、既存の位置共有アプリを両親に勧めようとすると、途端にハードルが上がります。「あなたの居場所を常に見られるようにして」——仲の良い親子であっても、そんなお願いはなかなかできません。

「今、家にいるのかどうか」——そのくらいの情報を、もっと気軽に共有できたら。お互いの日常を尊重しながら、それでもちゃんとつながれる方法はないだろうか。

監視するのも、されるのも、誰だって嫌なものです。「監視」という言葉自体が嫌ですし、される側にしてみれば、プライバシーへの配慮に欠ける印象を受けるかもしれません。それは子どもでも大人でも同じです。自分が監視される立場をリアルに想像すれば、そう思うものです。

「子どもを管理する」のではなく、「家族としてつながる」ことを考えたとき、今よりももう少しお互いを尊重できる、ゆるい新しいつながりの形があってもいいのではないか。Kiniaは、そんな考えから生まれました。

Problem

既存の位置共有アプリが抱える問題

開発を始めてから改めて調べてみると、安全上の理由から「子どもの見守り」には確かに大きな需要がありました。そして、そのためには詳細な位置情報の共有が必要な場面もあることが分かりました。

しかし、既存のアプリには構造的な問題があります。それは「見る側」と「見られる側」のニーズが対称ではないということです。

📡

見る側の心理

親は子どもの位置を常に詳しく知りたい。けれど、自分の位置は子どもにそこまで見せたくない。見守りたいけれど、見守られたくはない。

😣

見られる側の負担

24時間、位置を見られている感覚はストレスです。ほんの一世代前まで、こんな詳細に家族の行動を把握する術はありませんでした。便利な技術が、新しい息苦しさを生んでいます。

⚖️

不公平な関係

自分がその立場だったら、どう感じるか。居場所を常に見られているのに、相手の居場所は分からない。「あなたのためだから」と言われても、納得できないものは納得できない。——見る側と見られる側で共有のレベルが違う。この一方通行の関係は、家族の中にある信頼を、気づかないうちに削っていきます。

🚫

導入のハードル

「位置を常に共有して」とは、家族にすら言い出しにくい。導入のハードルが高すぎて、本当に見守りが必要な人——たとえば離れて暮らす高齢の親——にこそ、届いていません。

もちろん、現代は子どもが犯罪に巻き込まれるリスクもあり、安全のために位置を把握したい気持ちはよく分かります。見守ること自体が悪いわけではありません。

問題は「見守り方」にあるのではないか。お互いが負担に感じない距離感でつながれること。そして、いざという時にはしっかりとつながれる仕組み——その両立を目指したのが、Kiniaのコンセプトです。

Principle

Kiniaの原則：全員がフェアであること

従来の見守りアプリには、暗黙の前提がありました。「見守る側」は情報を受け取るだけ。「見守られる側」は一方的に情報を提供させられる。——これは本質的に、監視と被監視の関係です。

でも、自分がされる側になった時を想像してみてください。自分の居場所を常に誰かに見られている。その相手がどこにいるかは教えてもらえない。それを受け入れられますか。

言う側・する側にいると、その行為を正当化できる。「安全のため」「心配だから」「親として当然」。理屈は分かる。でも、やられている側は納得していない。納得のない関係は、いずれ壊れる。

だからKiniaは、「する側の論理」ではなく「される側の感覚」を設計の出発点にしています。

知りたいなら、自分も開示する。大人であろうが子供であろうが、全員が同じ条件で情報を共有する。自分がされて嫌なことを、立場や年齢を理由に他人に強いない。——それが、Kiniaが考える「フェアであること」です。

Concept

全世代に向けた「つながりの形」

子どもの自立心を尊重しながら親は安心したい。子世代は離れた親の暮らしを気にかけたい。夫婦はお互いの行動を干渉せずに、でも帰りが遅ければ心配したい。——家族の「つながりたい」には、世代ごとにまったく違う形があります。

Kiniaでは、アプリを入れた時点での位置共有を通知エリアの出入りだけに限定しています。「自宅」「学校」「会社」など、あらかじめ設定した場所への出入りだけが家族に届く。それ以外の行動は、誰にも見えません。

家族に届く通知の例
🏠 【OUT】太郎 → 自宅を出ました
🏫 【IN】太郎 → 学校に着きました
📚 【OUT】太郎 → 塾を出ました
🏠 【IN】太郎 → 自宅に着きました

メイン画面は地図ではなく、テキストベースのメンバー一覧です。「誰が、いつ、どこにいて、元気なのか」が一目で分かる。位置共有アプリとしては少し変わった設計ですが、これがKiniaの考える「ちょうどいい距離感」です。

もちろん、もっと詳しくつながりたい家族のために、すべての位置情報を共有する設定も用意しています。家族の形は一つではないから、つながり方も一つである必要はありません。

Solution

Kiniaの答え

お互いが負担に感じない距離感でつながれる。
見守る側と見守られる側に、上下関係を作らない。

  • ✓ 初期設定は通知エリアのみ — 設定するまで何も共有されない
  • ✓ 「自宅を出ました」「学校にいます」レベルの共有
  • ✓ 地図ではなくテキスト一覧がメイン画面
  • ✓ 必要な時だけ、一時的に詳細な位置を共有
  • ✓ お子さんの自立性と個性を尊重した設計

Security

管理・運営する我々ですら、データを直接見ても位置情報がわからない設計

位置情報は、とてもデリケートな情報です。それを誰かに預けるとき、「運営に見られているんじゃないか」「どこかで使われているんじゃないか」と不安に感じる方もいると思います。その気持ちは、当然のことです。

Kiniaはその不安に、仕組みで応えます。「あなたの位置情報は、運営を含め誰にも見えません」——これは言葉だけの約束ではありません。技術的に見ることができないよう、設計そのものに組み込んであります。

「サービスの管理者なら、データを見られるんじゃないの？」——そう思われるかもしれません。普通のサービスであれば、その通りです。しかしKiniaでは、管理者であっても中身を読み取ることができない仕組みになっています。

Kiniaでは、位置情報は合言葉（グループID）から作り出される鍵で暗号化されてからサーバーに保存されます。合言葉はサーバーには送信されず、あなたのスマートフォンにだけ保存されています。つまり、データの鍵を持っているのはあなたの家族だけ。合言葉を含めて、暗号を解くために必要な情報は、管理・運営する我々を含めて誰も把握することができません。

データベースそのものは、Googleが管理するクラウド基盤（Firebase）上にあり、通信はすべて暗号化されています。世界中の企業が利用しているGoogleのセキュリティ基盤で、不正なアクセスからしっかり守られています。

これは単なるセキュリティ対策ではなく、開発者としての約束です。利用者の個人情報に該当するものや、個人を特定できる情報を、そもそも持ちたくない。預からなければ、守る責任も、漏洩のリスクも生まれない。——大切な情報を預けてくださる方の不安に寄り添うために、「持たないこと」を選びました。運営する側にとっても、利用する側にとっても、それが最も誠実な設計だと考えています。

Googleのセキュリティ基盤

データベースへのアクセスは、Googleアカウントによる認証で保護されています。ログインしていない第三者がデータに触れることはできません。通信もすべて暗号化されており、途中で盗み見られる心配もありません。

位置情報の暗号化

サーバーに保存される位置情報は、合言葉（グループID）から作り出される鍵で暗号化されています。合言葉はサーバーに送信されないため、管理者であっても実際の位置を知ることはできません。

「誰が誰か」分からない設計

サーバー上にはランダムなIDだけが保存されています。家族の名前はあなたの端末にだけ保存され、サーバーには一切送信されません。データベースを直接見ても、どのIDが誰かなどの情報は一切わかりません。

「持たない」という安全

位置情報が意味を持つのは、合言葉を知っている家族の端末上だけです。サーバーは暗号化されたデータを中継するだけの存在。個人を特定できる情報をそもそも預からないことが、利用者にとっても運営にとっても、最も安全な形だと考えています。

More Features

位置共有だけじゃない、家族のための機能

👟

歩数と元気度

家族の今日の歩数と、そこから算出した「元気度」が分かります。位置を共有しなくても「今日も元気に動いてるな」と安心できます。

💬

ステータスコメント

「会議中」「外出中」「帰宅中」など、一言メッセージを設定できます。有効時間を選べるので、自動で消えます。

🆘

困っているアラート

緊急時にワンタップでSOSを家族全員に送信。プッシュ通知で即座に届きます。

💰

完全無料・広告なし

基本機能はすべて無料で利用できます。広告表示もありません。家族の安心にお金はかかりません。

小学校、中学校、高校——学生時代だけで終わるアプリではありません。
大学に進学しても、社会人になっても、家族が離れて暮らすようになっても。

フェアであることが、長く続く理由になる。

災害時の安否確認

将来的には、災害が起こった時にお互いの安否確認や
最終位置の情報だけは即時に共有できる仕組みを構築します。
普段は「ゆるいつながり」、いざという時は「確実につながる」。
それがKiniaの目指す未来です。

Name

「Kinia（キニア）」の由来

気にかける（Kini）＋近く（Near）

家族の帰りが遅いとき、ふと「どうしたのかな」と気になった（Kini）瞬間、
アプリを開けば家族の存在を近く（Near）に感じられる。

常に監視し合うのではなく、気になった時だけそっと見守る。
「緩やかで温かい距離感」を名前に込めました。
