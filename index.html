Story

なぜKinia（キニア）を作ったのか

世間一般に見れば、私は実家の家族とも義理の家族とも、かなり仲が良い方だと思っています。
これまでは別の位置共有アプリを使って、お互いの居場所を共有してきました。

ただ、使い続けるうちに、ある違和感が大きくなっていきました。

常にお互いの位置が詳細に見えるという関係は、
共有のレベルとして少し強すぎるのではないか。

比較的、適切な距離感を管理できている間柄であっても、常時の位置共有はギリギリのラインだと感じます。
私自身も、自分の移動が常に家族に見えていると思うと、どこか落ち着かない感覚がありました。

そして、自分の実家の両親にいたっては、位置共有の心理的ハードルが高すぎて、
アプリの紹介すらできていない状況でした。
「あなたの位置を常に見られるようにして」—— そんなことは、仲の良い親子でもなかなか言い出せません。

でも一方で、離れて暮らしていると気になることもあります。
「今日はちゃんと家にいるかな」「外出先で何かあっていないかな」
そういう、ふとした心配の瞬間。

「今、家にいるのかどうか」——
そのくらいの情報を、お互いにもっと緩く共有できたら、
より適切な距離感で付き合えるのではないか。

監視するのもされるのも、お互いに嫌なものです。
「子どもの見守り」という視点ではなく、「家族同士のつながり」を考えたとき、
もう少し距離を取った「ゆるいつながりの形」があるのではないか。
そう思ったのが、このアプリを作ろうとしたきっかけでした。

Problem

既存の位置共有アプリが抱える問題

調べていくうちに、やはり安全上の理由から「子どもの見守り」には一定の需要があり、
そこでは詳細な位置情報の共有が必要になることも分かってきました。

しかし、そこにはニーズの非対称性が存在します。

📡

監視する側の心理

主に親側は、常にリアルタイムで詳細な情報を見たい。
でも自分の情報は — 子どもに対して — ある程度ぼかして共有したい。

😣

見守られる側の負担

常に位置を見られている感覚はストレス。
昔はこれほど詳細に干渉する術がなかった。
今の技術が生んだ新しい問題です。

⚖️

不公平な非対称性

見る側と見られる側で共有レベルが異なる。
一方的な関係は、家族の信頼を静かに削っていきます。
この非対称性は、少し不公平だと感じました。

🚫

高すぎる心理的ハードル

親世代に「位置を常に共有しよう」とは言い出しにくい。
家族に勧めること自体が難しい。
結果として、本当に必要な人に届かない。

「監視」という言葉は語弊がありますが、
子どもの立場に寄り添うわけではないにしても、
一方で、現代は犯罪に巻き込まれないかといった心配もあります。

見守られる側が負担に感じないような見せ方や共有のされ方、
そして「いざという時」だけはしっかりと繋がれるような仕組みを
構築できないか —— それが、Kiniaのコンセプトです。

Concept

全世代に向けた「つながりの形」の定義

お子さん自身の自立性や個性、個人を尊重した上で、親御さんや、子世代から見た親世代、全世代に向けた「つながりの形」をここで定義したい —— そう考えています。

だからこそ、アプリインストールの初期値としての位置共有は、特定のエリア — 設定されたエリアに入った、あるいは出たといった情報 — に限定することを推奨しています。

共有される情報の例
🏠 【OUT】太郎 → 自宅を出ました
🏫 【IN】太郎 → 学校に着きました
📚 【OUT】太郎 → 塾を出ました
🏠 【IN】太郎 → 自宅に着きました

その場合、ユーザーインターフェースもマップ上に詳細な位置情報を表示することをメインとはしません。「誰が、いつ、どんな状態にあるか」「今その人はどこにいるのか、元気なのか」といった情報を一覧でパッと確認できるような、テキストベースの少し変わったインターフェースをコンセプトとしています。

もちろん、その世代や家族のつながりのあり方はそれぞれです。ニーズに合わせた使い方ができるよう、柔軟な設計になっています。

Solution

Kiniaの答え

見守られる側が負担に感じない共有のされ方。
そして「いざという時」だけはしっかりと繋がれる仕組み。

  • ✓ 初期設定は通知エリアのみ — 設定するまで何も共有されない
  • ✓ 「自宅を出ました」「学校にいます」レベルの共有
  • ✓ 地図ではなくテキスト一覧がメイン画面
  • ✓ 必要な時だけ、一時的に詳細な位置を共有
  • ✓ お子さんの自立性と個性を尊重した設計

How it works

届く通知は、これだけ

詳細な現在地ではなく、登録したエリアへの出入りだけ。
誰が、いつ、どこにいるか — それだけで十分な安心があります。

Security

運営すら位置情報を見られない設計

Kiniaでは、サーバーに保存される位置情報は独自の暗号化技術でオフセット処理されています。 万が一データベースが流出しても、実際の座標を復元することは不可能です。

🔐

座標オフセット暗号化

全ての位置情報は、端末にのみ保存される「合言葉」から導出したオフセットでシフトされてからサーバーに保存されます。合言葉はサーバーに送信されないため、運営を含め誰も実際の位置を逆算できません。

👤

個人名をサーバーに保存しない

サーバー上にはランダムなIDのみが保存されます。家族の表示名はあなたの端末にだけ保存され、サーバーには送信されません。データベースを見ても「誰が誰か」は分かりません。

🔑

グループIDのハッシュ化

グループへの参加に使う合言葉も、暗号学的ハッシュ関数で一方向変換されてからサーバーに保存されます。サーバーから合言葉を逆算することは不可能です。

🛡️

エンドツーエンドの安全性

位置情報が意味を持つのは、合言葉を知っている家族の端末上だけです。サーバーは暗号化されたデータを中継するだけの存在です。

More Features

位置共有だけじゃない、家族のための機能

👟

歩数と元気度

家族の今日の歩数と、そこから算出した「元気度」が分かります。位置を共有しなくても「今日も元気に動いてるな」と安心できます。

💬

ステータスコメント

「会議中」「外出中」「帰宅中」など、一言メッセージを設定できます。有効時間を選べるので、自動で消えます。

🆘

困っているアラート

緊急時にワンタップでSOSを家族全員に送信。プッシュ通知で即座に届きます。

💰

完全無料・広告なし

基本機能はすべて無料で利用できます。広告表示もありません。家族の安心にお金はかかりません。

Name

「Kinia（キニア）」の由来

気にかける（Kini）＋近く（Near）

家族の帰りが遅いとき、ふと「どうしたのかな」と気になった（Kini）瞬間、
アプリを開けば家族の存在を近く（Near）に感じられる。

常に監視し合うのではなく、気になった時だけそっと見守る。
「緩やかで温かい距離感」を名前に込めました。

ゆるいつながり、始めませんか

Kiniaは現在ベータテスト中です。
あなたの家族の「ちょうどいい距離感」を、一緒に見つけましょう。
